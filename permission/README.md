# Security and Permission - Thinreports Examples

## How to run

Bundle in this directory:

    % bundle install

Then execute `permisson.rb`:

    % bundle exec ruby permission.rb

`result.pdf` will be created.

## Requirements

  * ruby >= 1.9.3
  * thinreports >= 0.8.0
