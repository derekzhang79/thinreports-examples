# TextBlock Tool - Thinreports Examples

## How to run

Bundle in this directory:

    % bundle install

Then execute `text_block.rb`:

    % bundle exec ruby text_block.rb

`result.pdf` will be created.

## Requirements

* ruby >= 1.9.3
* thinreports 0.9.0
