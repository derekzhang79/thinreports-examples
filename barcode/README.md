# Barcode - Thinreports Examples

## How to run

Bundle in this directory:

    % bundle install

Then execute `barcode.rb`:

    % bundle exec ruby barcode.rb

`result.pdf` will be created.

## Requirements

* ruby >= 1.9
* Runtime dependencies
  * thinreports >= 0.7.7
  * Barby
  * rQRCode
  * ChunkyPNG
