# Use External-Character - Thinreports Examples

## How to run

Bundle in this directory:

    % bundle install

Then execute `eudc.rb`:

    % bundle exec ruby eudc.rb

`result.pdf` will be created.

## How to get EUDC.ttf

  1. Windows の外字エディタで外字を作る
  2. 作成された `EUDC.TTE` をコピー
  3. 拡張子を `ttf` へ変更

## Requirements

  * ruby >= 1.9.3
  * thinreports >= 0.8.0
